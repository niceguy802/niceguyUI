# niceguyUI
自用UI组件

# 安装
```bash
npm install niceguyui
```
# 构建组件库
```bash
npm run build:components
```

# 版本升级
## 补丁版本 1.0.0 -> 1.0.1
```bash
npm version patch
```
## 小版本 1.0.0 -> 1.1.0
```bash
npm version minor
``` 
## 大版本 1.0.0 -> 2.0.0
```bash
npm version major
``` 

# 发布到npm
```bash
npm publish --access public
```
