# Button 按钮组件

按钮组件示例，展示基础用法、类型、大小、禁用状态等。

---

## 基础按钮

:::demo 基础按钮
<template>
  <el-button type="primary">Primary</el-button>
  <el-button type="success">Success</el-button>
  <el-button type="warning">Warning</el-button>
  <el-button type="danger">Danger</el-button>
</template>
:::

---

## 按钮大小

:::demo 按钮大小
<template>
  <el-button type="primary" size="large">Large</el-button>
  <el-button type="primary" size="medium">Medium</el-button>
  <el-button type="primary" size="small">Small</el-button>
</template>
:::

---

## 禁用状态

:::demo 禁用按钮
<template>
  <el-button type="primary" disabled>Primary Disabled</el-button>
  <el-button type="success" disabled>Success Disabled</el-button>
</template>
:::
